2. steal the limelight from sb -> 抢走某人的风采/风头
3. took a nosedive -> 大跌
4. pick oneself up -> 跌倒后爬起来
5. resort to -> 诉诸于
6. lag behind -> 落后
7. work towards the same goal -> 向着同一个目标去努力
8. infuse sb with sth -> 灌输某人（某种思想/理念/态度）
9. root for -> 为...加油，助威
